The Web & HTML:
The Basics

World Wide Web

The World Wide Web (www) is a collection of HTMLs (HyperText Markup Language).
HTMLs are linked through HyperLinks.

HTML

  1. Components
    • text content: what you see
    • markup: what it looks like (e.g. <b></b>)
    • references to other documents: images, videos, etc.
    • links to other pages
  2. HTML attributes
    • <tag attribute="value">content</tag>
    • <a href="link"></a>
    • <img src="url" alt="text"> (VOID tag: no content, no closing tag)
  3. Tags
    • <br>: break (inline)
    • <p>content</p>: paragraph (block)
  4. Elements
    • <span>text</span> (inline)
    • <div>text</div> (block)

Template?

A library to build complicated strings(e.g. HTML)

Jinja2 (built into Google App Engine)

Documentation: jinja.pocoo.org

  1. Getting environment
    environment = jinja2.environment(parameters)
  2. Getting template
    template=environment.get_template(file_name.html)
  3. Variable substitution
    [Inside HTML]
    {{ variable }}

    [Server code]
    template.render({variable: value})
  4. Control flow
    {% if name == "steve" %}
Hello, Steve!
{% else %}
who are you?
{% end if %}
  5. For loop syntax
    {% for statement %} {% endfor }
